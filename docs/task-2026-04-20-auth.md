## 登录流程

```
用户输入邮箱/密码
       ↓
POST /api/auth/login
       ↓
signIn('credentials', { email, password })
       ↓
authorize() → 查询数据库验证密码
       ↓
返回 JWT Session
       ↓
前端获取 session → 显示用户信息
```

---

## 登出流程

```
点击"退出登录"
       ↓
POST /api/auth/logout
       ↓
清除 cookies + signOut()
       ↓
跳转到首页
```
